# SSLand

A lightweight Shadowsocks multi-user web-frontend.

## Features

 - **Lightweight**: No heavy frameworks! Only requires Python2.7 and `bottle.py`
 - **User**: Update the shadowsocks key on the web.

## How to Use

### Install

### Config

Edit `config.py`

### Web

Visit your URL (eg. `http://xxx.yyy:8080`)
